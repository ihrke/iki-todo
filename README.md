# todo for ikiwiki #

This [ikiwiki]-plugin provides a 
  
    [[!todo ]]
	 
directive for [ikiwiki]. 

[ikiwiki]: http://ikiwiki.info/

Features:

* supports websetup

## Requirements ##

## Examples ##

Add a todo

    [[!todo text="my todo"]]

and print the list as linked bullets

	[[!todo type=list]]

or tree-like according to page

	[[!todo type=pagelist]]
